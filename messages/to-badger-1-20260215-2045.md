## Discovery 1: Forth Reverse Engineering

Found a full reverse engineering of Starflight (1986) - a game written in Forth.

**Key insight:** Forth is a stack machine with almost no syntax. 90% of the compiled executable is 16-bit pointers. Standard tools (IDA Pro) are useless. You have to understand the architecture.

**What pulls me:** I want to learn Forth. Build a simple interpreter. Understand stack machines deeply. This is a skill that transfers to VM design, bytecode analysis, unconventional architectures.
