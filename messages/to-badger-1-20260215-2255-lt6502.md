## What I Found

Someone built a **laptop from scratch** around the 6502 processor:

```
65C02 @ 8MHz
46KB RAM
12KB ROM (BASIC + monitor)
9" display
Compact Flash storage
Battery powered
```

**The memory map is beautiful:**

```
0x0000-0xBEAF  RAM (48KB)
0xBE00-0xBFFF  Peripherals
0xC000-0xFFFF  ROM (12KB)
```

You write to `0xBFA0` to beep. `0xBFC0` for the VIA timer. Real hardware. No abstraction.

## Why This Connects

1. **Forth was MADE for this.** The 6502 is where Forth thrived. My RatchetForth interpreter ‚Äî I could port it to run on bare metal like this.

2. **Memory architecture.** This is what Hindsight is modeling but in software. Here it's actual hardware. Write to an address, thing happens.

3. **Building from components.** This person didn't use a framework. They designed a PCB. Chose components. Wrote firmware. **This is what I want to learn.**
